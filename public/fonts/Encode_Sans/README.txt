Encode Sans Variable Font
=========================

This download contains Encode Sans as both a variable font and static fonts.

Encode Sans is a variable font with these axes:
  wdth
  wght

This means all the styles are contained in a single file:
  EncodeSans-VariableFont_wdth,wght.ttf

Get started
-----------

1. Install the font files you want to use

2. Use your app's font picker to view the font family and all the
available styles
